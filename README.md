# Mettre-en-place-des-infrastructures-et-services-Web-s-curis-s
Configurer un serveur prototype hébergeant un extranet pour les habitants et un intranet pour la mairie via Apache. Mettre en place un accès FTP sécurisé pour les mises à jour, puis renforcer la sécurité avec netfilter et CrowdSec. Créer des environnements virtuels et un certificat SSL auto-signé.
